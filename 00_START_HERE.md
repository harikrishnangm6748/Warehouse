## 🎯 User Workflows

### Employee Workflow
1. Login as EMPLOYEE
2. Dashboard shows statistics
3. Create Order → Add items → Submit
4. My Orders → View created orders
5. Click order → See details & timeline
6. Track progress in real-time

### Warehouse Workflow
1. Login as WAREHOUSE
2. Orders Queue → See all orders
3. Inventory Check → Reserve items
4. Dispatch Order → Create shipments
5. Track shipment progress

### Operations Workflow
1. Login as OPERATIONS
2. Shipments Queue → Monitor all
3. Shipment Details → See tasks
4. Add exceptions as needed
5. Track delivery progress
